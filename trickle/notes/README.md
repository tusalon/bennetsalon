# Uñas Perfectas - Sistema de Reservas

Aplicación web para la gestión de turnos de un centro de manicuría.

## Características
- Selección de servicios con duraciones específicas.
- Calendario interactivo (Año 2026).
- Generación inteligente de horarios disponibles basándose en la duración del servicio y turnos existentes.
- Integración con WhatsApp para notificaciones.
- Panel de administración para gestionar turnos.

## Estructura
- `index.html`: Página principal de reservas para clientes.
- `admin.html`: Panel de administración para el dueño.
- `app.js`: Lógica principal de la app de reservas.
- `admin-app.js`: Lógica del panel de administración.
- `utils/`: Funciones auxiliares y lógica de negocio.
